Термины

Бенчмарк — сравнительное тестирование нескольких методов или моделей на одном наборе данных с целью выбора наилучшего решения по заданным метрикам качества.
Векторный эмбеддинг (вектор представления) — числовое представление текста или другого объекта в многомерном пространстве, при котором семантически близкие объекты располагаются вблизи друг друга.
Детекция объектов — задача компьютерного зрения, заключающаяся в обнаружении и локализации объектов заданных классов на изображении с помощью ограничивающих прямоугольников (bounding box).
Инференс — процесс применения обученной модели машинного обучения к новым входным данным для получения предсказания.
Компонентный состав газа — перечень составляющих топливного газа (метан, этан, пропан и др.) с указанием их концентраций в объёмных, молярных или массовых долях.
Косинусное сходство — мера схожести двух векторов, вычисляемая как косинус угла между ними; принимает значения от −1 до 1, где 1 означает полное совпадение направления векторов.
Кросс-валидация (перекрёстная проверка) — метод оценки качества модели машинного обучения, при котором данные разбиваются на несколько фолдов; модель обучается на одних фолдах и проверяется на остальных, после чего результаты усредняются.
Меркаптановая сера — суммарное содержание серы в составе меркаптановых соединений топливного газа, выраженное в мг/нм³; нормируется ГОСТ 5542 как не более 36 мг/нм³.
Молярная доля — отношение количества молей данного компонента к общему количеству молей всех компонентов смеси, выраженное в процентах.
Нормализованное токенное сравнение — метод сопоставления строк, при котором исходные строки разбиваются на отдельные токены (слова), после чего сравниваются множества токенов без учёта порядка их следования.
Объёмная теплотворная способность — количество теплоты, выделяемое при сгорании единицы объёма топливного газа; нормируется ГОСТ 5542 как не менее 7600 ккал/м³.
Пайплайн — последовательная цепочка этапов обработки данных, в которой результат каждого этапа служит входом для следующего.
Парсинг — автоматический разбор и извлечение структурированных данных из неструктурированного или слабоструктурированного источника (PDF-файла, OCR-результатов и т.п.).
Паспорт качества / протокол испытаний — документ, содержащий результаты лабораторного анализа состава топливного газа; входной документ для системы.
Сессия — единица обработки в системе, объединяющая один или несколько протоколов испытаний (до трёх), загруженных пользователем за один раз для совместного анализа.
Семантический поиск — поиск по смысловой близости запроса и документов, реализованный через сравнение их векторных представлений, в отличие от ключевого поиска по точному совпадению слов.
Температура точки росы — температура, при которой пары углеводородов или воды в топливном газе начинают конденсироваться при данном давлении; ГОСТ 5542 требует подогрева газа до температуры на 20 °C выше точки росы.
Трансформер (Transformer) — архитектура глубокого обучения, основанная на механизме самовнимания (self-attention); применяется в большинстве современных моделей обработки текста и изображений.
Union-Find (система непересекающихся множеств) — алгоритмическая структура данных, поддерживающая операции объединения элементов в группы и определения принадлежности элемента к группе за почти константное время; используется в системе для объединения OCR-блоков в строки.
Few-shot prompting (обучение на примерах) — подход к работе с языковыми моделями, при котором в запрос включается несколько примеров — эталонных пар «вход — ожидаемый выход», позволяющих модели адаптироваться к конкретной задаче без дообучения.
Фолд — одна из равных частей, на которые разбивается набор данных при кросс-валидации; в системе используется 5-фолдовая кросс-валидация.
ChromaDB — векторная база данных с открытым исходным кодом, специализированная на хранении и поиске векторных эмбеддингов по мерам схожести; используется в системе для поиска аналогов протоколов.
Число Воббе — характеристика взаимозаменяемости горючих газов, определяемая как отношение высшей теплотворной способности к квадратному корню из относительной плотности газа; нормируется ГОСТ 5542 в диапазоне 41,2–54,5 МДж/м³.
